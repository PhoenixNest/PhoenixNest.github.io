Introduction

2020年7月10日，Canonical开始将Flutter引入Ubuntu Linux，作为一个热(wo)爱(ai)学(mo)习(yu)的苦逼底层程序猿，也将计划已久的Flutter学习计划列进了日程。

Flutter作为新时代跨平台语言的首选之一，也不断地在向世界展示其强大性与易用性：基于Material Design的组件库，基于Dart的前端快速开发语言，热更新，原生体验。Flutter能快速地在桌面端(Windows、macOS、Linux)与移动端(IOS，Android)之前进行部署构建，打造高质量原生用户界面。

本篇将记录我在使用Ubuntu Linux时的一些笔记与问题解决分享，有关Flutter的学习笔记后续将会新开博文更新。
